Desafios Superados

Este projeto foi construído do zero — sem experiência prévia criando infraestrutura AWS. Cada etapa foi um aprendizado deliberado.

1. Configuração AWS do zero

IAM policies, SAM template, deploy automatizado — tudo aprendido e configurado pela primeira vez. Nunca havia criado recursos AWS from scratch.

2. .NET 8 rodando no Lambda

A maioria dos exemplos no GitHub usa Python/JS. Provar que .NET é cidadão de primeira classe no Lambda exigiu pesquisa, ajustes e benchmarks.

3. Integração com Rekognition (IA)

Implementar auto-tagging por IA sem treinamento de modelo — usando a API DetectLabels para classificação automática de imagens.

4. Client desktop com .NET MAUI 10

Construir um app cross-platform real consumindo a API serverless, com MVVM, navegação Shell e suporte a temas claro/escuro.

5. Ambiente local com LocalStack

Emulação de S3 + DynamoDB localmente para desenvolvimento sem custo e sem dependência de conexão com a nuvem.

6. Pipeline de evidências automatizado

Scripts PowerShell para deploy + coleta de evidências (health, listagem, stats, testes unitários) — tudo reproduzível.
