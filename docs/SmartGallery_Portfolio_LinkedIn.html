Integrações AWS em detalhe

Auto-Tagging por Inteligência Artificial

Ao enviar uma imagem, o Amazon Rekognition analisa automaticamente o conteúdo e detecta objetos, cenas e conceitos. Tags como beach, coast, driftwood, landscape são geradas sem nenhuma intervenção manual.

A tela de detalhes exibe: imagem em alta resolução, todas as tags (manuais + IA), metadados completos — resolução, tamanho, formato e data de upload.

Rekognition DetectLabels S3 Pre-signed URL .NET MAUI MVVM

Dashboard de Analytics em tempo real

Painel com KPIs consolidados: total de imagens na galeria, armazenamento utilizado no S3, distribuição por formato de arquivo e ranking das tags mais populares — incluindo as geradas automaticamente pela IA.

Os dados são agregados no servidor (Lambda) e consumidos pelo client MAUI via endpoint /api/imagens/stats.

Plataformas

Por que .NET no Lambda? A comunidade AWS é dominada por Python e JavaScript. Escolher .NET 8 demonstra versatilidade, performance e preparo para ambientes corporativos. Este é um dos raros projetos de referência combinando .NET + Lambda + Rekognition.

Jornada pessoal
Desafios superados
Construído do zero, sem experiência prévia criando infraestrutura AWS.

Configuração AWS do zero

IAM policies, SAM template, buckets, tabelas — tudo aprendido e configurado pela primeira vez. Nunca havia criado recursos AWS from scratch.

.NET 8 rodando no Lambda

Provar que .NET é cidadão de primeira classe no Lambda exigiu pesquisa, ajustes de runtime e benchmarks de cold start.

Integração com Rekognition (IA)

Auto-tagging sem treinamento de modelo — usando DetectLabels para classificação automática de imagens.

Client desktop com .NET MAUI 10

App cross-platform real consumindo API serverless, com MVVM, navegação Shell e suporte a temas.

Ambiente local com LocalStack

Emulação de S3 + DynamoDB localmente para desenvolvimento sem custo e sem dependência de conexão com a nuvem.

Pipeline de evidências automatizado

Scripts PowerShell para deploy, coleta de evidências e snapshots — tudo reproduzível e documentado.
